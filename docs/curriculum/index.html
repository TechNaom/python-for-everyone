Curriculum Roadmap

Rebuilt from ten years of real classroom teaching material into 33 chapters, grouped into modules. Each chapter is fully piloted (lesson, quiz, exercises, interview questions, project) before the next one starts, and every module ends with a written exam.

As of Chapter 25, every new chapter is built GenAI-driven by default: a thought-process + prompt box after every worked example (the reasoning behind the code, then a space to write your own AI prompt and compare it against a real one), and an "AI-paired" critique exercise on every project, where a realistic AI-generated solution has genuine, subtle bugs planted in it to find. Chapter 25 (Memory Management) is the reference implementation; a retrofit pass to bring Chapters 1–24 up to the same standard is in progress.

Format every chapter follows

Hook → plain-language concept → "What is...?" callouts → annotated code (with a live in-browser "Run Code" playground for examples that don't need input()) → optional "Go Deeper" box → a quick recap after each sub-topic → a full Points to Remember at the end → an interactive fill-in-the-blank quiz (with a progress bar that marks the chapter complete in your browser) → exercises (including a "Debug the Code" task) → interview questions → a real, resume-worthy project. Every module of 4-6 chapters ends with a written exam. See CONTRIBUTING.md for the exact file structure and content standards.
