# Python for Everyone — Curriculum Map

This is a **draft roadmap**. Only Chapter 1 is built so far — the course is
being developed one chapter at a time, with each new chapter validated
against the format Chapter 1 established before moving to the next. Exact
chapter count, order, and titles may still change.

## Format every chapter follows

Established and validated in Chapter 1 — see [`CONTRIBUTING.md`](../../CONTRIBUTING.md)
for the exact file structure and content standards each new chapter must
match: hook, plain-language concept, "What is...?" callouts, annotated code,
optional "Go Deeper" box, Points to Remember, an interactive self-checking
quiz, exercises, interview questions, and a real project.
