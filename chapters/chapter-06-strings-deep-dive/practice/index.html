Chapter 6 Practice Bank 50-65 min, hands-on

Practice Bank: Strings Deep Dive

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Uses everything from Chapters 1-5 plus this chapter's own toolkit: len(), indexing/slicing, the string methods from the lesson, f-strings, and iterating a string or a .split() result with a bare for-loop. No lists/dicts, comprehensions, functions, or try/except yet.

On this page 1. Indexing & Slicing 2. Common String Methods 3. f-strings 4. Immutability & Identity 5. Searching & Validating 6. Real-World Text Processing
How to run these Download starter.py below, then run it from a terminal with python3 starter.py.
