def is_palindrome(s):

3 Count the Vowels Easy

Write a function count_vowels(s) that returns how many of the characters in s are vowels (a, e, i, o, u), counting both upper and lowercase.

Constraints: s may be empty, and may mix upper/lowercase letters.
Examples:
count_vowels("hello") → 2
count_vowels("PYTHON") → 1
count_vowels("") → 0
challenge_3.py
Show Solution
solution_3.py 
def count_vowels(s):
    count = 0
    for char in s.lower():
        if char in "aeiou":
            count = count + 1
    return count

6 Capitalize Each Word Medium

Write a function capitalize_words(sentence) that returns sentence with the first letter of every word uppercase and the rest of each word lowercase, words separated by a single space. Build the result yourself with .split() and a for-loop — don't rely on .title() alone (it mishandles things like apostrophes).

Constraints: sentence may be empty, a single word, or contain extra spaces between words (which .split() already collapses for you).
Examples:
capitalize_words("hello world") → "Hello World"
capitalize_words("PYTHON is FUN") → "Python Is Fun"
capitalize_words("") → ""
challenge_6.py
Show Solution
solution_6.py 
def capitalize_words(sentence):
    result = ""
    first = True
    for word in sentence.split():
        piece = word[0].upper() + word[1:].lower()
        if first:
            result = piece
            first = False
        else:
            result = result + " " + piece
    return result

7 Mask a Card Number Medium

Write a function mask_string(s, keep) that replaces every character in s except the last keep characters with "*" — the way a receipt hides most of a credit card number.

Constraints: keep may be 0 (mask everything) or greater than len(s) (nothing to mask — return s unchanged).
Examples:
mask_string("4111111122223333", 4) → "************3333"
mask_string("hello", 2) → "***lo"
mask_string("hi", 4) → "hi"
challenge_7.py
Show Solution
solution_7.py 
def mask_string(s, keep):
    masked_len = len(s) - keep
    if masked_len < 0:
        masked_len = 0
    visible_start = len(s) - keep
    if visible_start < 0:
        visible_start = 0
    return "*" * masked_len + s[visible_start:]

8 Anagram Check Medium

Write a function is_anagram(a, b) that returns True if a and b use exactly the same letters the same number of times (ignoring case and spaces), and False otherwise. Solve it using only .lower(), .replace(), len(), .count(), and a for-loop — no sorted(), lists, or sets.

Constraints: comparisons are case-insensitive and ignore spaces; both strings may be empty.
Examples:
is_anagram("listen", "silent") → True
is_anagram("hello", "world") → False
is_anagram("Dormitory", "Dirty Room") → True
challenge_8.py
Show Solution
solution_8.py 
def is_anagram(a, b):
    a_clean = a.lower().replace(" ", "")
    b_clean = b.lower().replace(" ", "")
    if len(a_clean) != len(b_clean):
        return False
    for ch in a_clean:
        if a_clean.count(ch) != b_clean.count(ch):
            return False
    return True
