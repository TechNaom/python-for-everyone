1 Refcount Change on Alias Easy

Constraints: obj can be any object (a list, string, or number).
Examples:
refcount_change_on_alias([1, 2, 3]) → 1

import sys

def refcount_change_on_alias(obj):
    before = sys.getrefcount(obj)
    alias = obj
    after = sys.getrefcount(obj)
    return after - before

3 Safe Append (Fix the Default) Easy

Write a function safe_append(item, items=None) that correctly avoids the mutable default argument gotcha: if items is not given, start from a fresh empty list every call, append item, and return the list.

Constraints: item can be any value; items, if given, is a list.
Examples:
safe_append("a") → ["a"]
safe_append("b") → ["b"] (not ["a", "b"])
challenge_3.py
Show Solution
solution_3.py 
def safe_append(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items

4 Simulate the Shared-Default Bug Medium

Write a function append_n_times_broken(n) that simulates the classic mutable default argument bug directly: define a local helper def _broken(item, cart=[]) inside append_n_times_broken, call it n times in a row with items 0, 1, 2, ..., n-1 (not resetting cart between calls), and return the final call's result as a plain list.

Constraints: n is a non-negative int.
Examples:
append_n_times_broken(3) → [0, 1, 2] (every call shared the same growing list)
append_n_times_broken(1) → [0]
challenge_4.py
Show Solution
solution_4.py 
def append_n_times_broken(n):
    def _broken(item, cart=[]):
        cart.append(item)
        return cart

    result = []
    for i in range(n):
        result = _broken(i)
    return result

5 Slotted vs. Regular Instance Size Medium

Write a function is_slotted_smaller(x, y) that builds two classes internally — one regular, one using __slots__ = ("x", "y") — each with matching __init__(self, x, y), creates one instance of each with the given values, and returns True if the slotted instance's sys.getsizeof() plus nothing else is strictly smaller than the regular instance's sys.getsizeof() plus its __dict__'s sys.getsizeof() combined.

Constraints: x and y are plain values (numbers or strings).
Examples:
is_slotted_smaller(1, 2) → True
challenge_5.py
Show Solution
solution_5.py 
import sys

def is_slotted_smaller(x, y):
    class Regular:
        def __init__(self, x, y):
            self.x = x
            self.y = y

    class Slotted:
        __slots__ = ("x", "y")

        def __init__(self, x, y):
            self.x = x
            self.y = y

    r = Regular(x, y)
    s = Slotted(x, y)
    regular_total = sys.getsizeof(r) + sys.getsizeof(r.__dict__)
    slotted_total = sys.getsizeof(s)
    return slotted_total < regular_total

7 Is the Generator Lighter? Medium

Write a function is_generator_lighter(n) that uses tracemalloc to compare, at the moment of creation, the peak memory of a list comprehension [i for i in range(n)] against the equivalent generator expression (i for i in range(n)). Return True if the generator's peak is strictly less than the list's peak, else False. Measure the generator's memory before consuming it.

Constraints: n is a positive int.
Examples:
is_generator_lighter(1000) → True
challenge_7.py
Show Solution
solution_7.py 
import tracemalloc

def is_generator_lighter(n):
    tracemalloc.start()
    lst = [i for i in range(n)]
    _, list_peak = tracemalloc.get_traced_memory()
    tracemalloc.stop()

    tracemalloc.start()
    gen = (i for i in range(n))
    _, gen_peak = tracemalloc.get_traced_memory()
    tracemalloc.stop()

    return gen_peak < list_peak
