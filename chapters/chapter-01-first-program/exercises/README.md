# Chapter 1 Exercises: Your First Python Program

These exercises use only what Chapter 1 covered: `print()`, variables, basic
types (string, int, float), `input()`, and joining text with `+` / `str()`.

## How to run

You'll need Python 3 installed. Check with:

```bash
python3 --version
```

Then run the starter file:

```bash
python3 starter.py
```

## Exercise 1 — Say hello, your way

Open `starter.py`. Find `# TODO 1` and print a greeting of your choice using
`print()`.

## Exercise 2 — Store your details

Find `# TODO 2`. Create three variables: `city` (a string), `favorite_number`
(an int), and `temperature_today` (a float). Print all three.

## Exercise 3 — Ask, don't tell

Find `# TODO 3`. Use `input()` to ask the user for their favorite color, store
it in a variable, then print a sentence that includes their answer.

## Exercise 4 — Fix the type error

Find `# TODO 4`. There's a line that tries to join a string with a number
directly and will crash with a `TypeError`. Fix it using `str()`, the way
Chapter 1's "Go Deeper" box explained.

## Checking your work

There's no automated grader for this exercise — that's intentional at this
stage. Compare your output against `solution.py` (run it with
`python3 solution.py`) to see one correct way of doing it. Your wording and
variable choices don't need to match exactly — the goal is that your program
runs without errors and does what each TODO asks.
