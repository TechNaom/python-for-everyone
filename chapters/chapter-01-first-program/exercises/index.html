Chapter 1 Exercises 15 min, hands-on

Exercises: Your First Python Program

These exercises use only what Chapter 1 covered: print(), variables, basic types (string, int, float), input(), and joining text with + / str().

How to run these You'll need Python 3 installed (check with python3 --version). Download starter.py below, then run it from a terminal with python3 starter.py.
starter.py solution.py (reference)

The four tasks

1 Say hello, your way

Open starter.py. Find # TODO 1 and print a greeting of your choice using print().

2 Store your details

Find # TODO 2. Create three variables: city (a string), favorite_number (an int), and temperature_today (a float). Print all three.

3 Ask, don't tell

Find # TODO 3. Use input() to ask the user for their favorite color, store it in a variable, then print a sentence that includes their answer.

4 Fix the type error

Find # TODO 4. There's a line that tries to join a string with a number directly and will crash with a TypeError. Fix it using str(), the way Chapter 1's "Go Deeper" box explained.
