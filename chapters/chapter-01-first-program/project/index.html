Chapter 1 Project 20 min build

Personal Greeting Generator

A small, complete, fun program you can actually show someone — built entirely from what Chapter 1 taught: print(), variables, input(), and joining text.

A Mad-Libs-style program that asks the person a few quick questions, then generates a silly, personalized greeting card using their answers. Every run produces a different result because it's built from their input, not fixed text.

Example run

Output 
What's your name? Asha
What's your favorite animal? penguin
What's your favorite food? mango
Pick a number from 1 to 100: 42

=== Your Personalized Greeting ===
Dear Asha,

Congratulations! You have been officially named
Grand Ambassador of Penguins, Protector of Mango,
and Keeper of the Sacred Number 42.

May your day be as wonderful as you are.

Build it

How to run Download starter.py below and run python3 starter.py. Fill in the # TODO sections to build your own version. Want to see one finished version first? Download and run solution.py.

Ideas to make it your own (optional stretch goals)

These go beyond Chapter 1's material, so don't worry if they don't make sense yet — they're here for anyone who wants to peek ahead:

  • Ask for a 5th piece of information and work it into the message.
  • Print the greeting inside a decorative border made of * or - characters.
  • Ask the person's age, and add a joke that depends on whether they're above or below a certain number (this needs if, which is Chapter 4 — totally fine to leave for later).

Why this project matters

This is intentionally small, but it proves something important: with just five ideas — printing, variables, types, asking questions, and joining text — you can already build something that feels personal and complete. That's the whole spirit of this course: real, working things, one solid layer at a time.
