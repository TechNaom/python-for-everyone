1 Total Revenue Easy

Write a function total_revenue(products, revenues) that builds a DataFrame from the two plain Python lists (columns "product" and "revenue") and returns the total of the revenue column as a plain Python int.

Constraints: products and revenues are same-length lists (possibly empty); revenue values are ints.
Examples:
total_revenue(["Widget", "Gadget"], [1200, 950]) → 2150
total_revenue([], []) → 0
challenge_1.py
Show Solution
solution_1.py 
import pandas as pd

def total_revenue(products, revenues):
    df = pd.DataFrame({"product": products, "revenue": revenues})
    return int(df["revenue"].sum())

Constraints: products and revenues are same-length lists.
Examples:

import pandas as pd

def filter_high_revenue(products, revenues, threshold):
    df = pd.DataFrame({"product": products, "revenue": revenues})
    return df[df["revenue"] > threshold]["product"].tolist()

3 Average Revenue Per Product Medium

Write a function average_revenue_per_product(products, revenues) that builds a DataFrame, groups by product, and returns the average revenue per product as a plain Python dict mapping product name (str) to average revenue (float).

Constraints: products may repeat; revenues is the same length.
Examples:
average_revenue_per_product(["Widget", "Gadget", "Widget"], [1000, 500, 2000]) → {"Widget": 1500.0, "Gadget": 500.0}
challenge_3.py
Show Solution
solution_3.py 
import pandas as pd

def average_revenue_per_product(products, revenues):
    df = pd.DataFrame({"product": products, "revenue": revenues})
    result = df.groupby("product")["revenue"].mean()
    return {k: float(v) for k, v in result.items()}

Constraints: n is a positive int, no greater than the number of products.
Examples:

import pandas as pd

def top_n_products(products, revenues, n):
    df = pd.DataFrame({"product": products, "revenue": revenues})
    return df.sort_values("revenue", ascending=False).head(n)["product"].tolist()

6 Count Missing Revenue Values Easy

Write a function count_missing_revenue(revenues) that builds a one-column DataFrame from revenues (a list that may contain None values) and returns how many values are missing, as a plain Python int.

Constraints: revenues is a list of numbers and/or None.
Examples:

import pandas as pd

def count_missing_revenue(revenues):
    df = pd.DataFrame({"revenue": revenues})
    return int(df["revenue"].isna().sum())

import pandas as pd

def fill_missing_with_zero(revenues):
    df = pd.DataFrame({"revenue": revenues})
    df["revenue"] = df["revenue"].fillna(0)
    return df["revenue"].tolist()

8 Row at a Position Easy

Write a function row_at_position(products, revenues, position) that builds a DataFrame and uses .iloc[] to return the row at integer position position as a plain Python dict with keys "product" (str) and "revenue" (int).

Constraints: position is a valid row position.
Examples:
row_at_position(["Widget", "Gadget"], [1200, 950], 1) → {"product": "Gadget", "revenue": 950}
challenge_8.py
Show Solution
solution_8.py 
import pandas as pd

def row_at_position(products, revenues, position):
    df = pd.DataFrame({"product": products, "revenue": revenues})
    row = df.iloc[position]
    return {"product": row["product"], "revenue": int(row["revenue"])}
