Practice Bank: Pandas for Data Analysis

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Every task here uses real pandas directly. No import beyond pandas/numpy/io/json/os/re/math/datetime/random/csv/threading/concurrent.futures.
