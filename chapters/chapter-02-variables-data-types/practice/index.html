Practice Bank: Variables & Data Types

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Still uses only print(), variables, input(), string concatenation with + / str(), identifier naming, dynamic typing, type(), id(), and the conversion functions int(), float(), str(), bool(), complex().

On this page 1. Naming variables properly 2. Python decides the type for you 3. type() and id() 4. Converting between types
How to run these Download starter.py below, then run it from a terminal with python3 starter.py.
