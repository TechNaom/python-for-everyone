# Chapter 2 Project: Everyday Unit Converter

A genuinely useful little tool built from variables, types, and
conversion — the kind of script you'd actually keep around.

## What you'll build

A converter that takes a Celsius temperature, a kilometer distance, and a
kilogram weight, and prints their Fahrenheit/Kelvin, mile, and pound
equivalents.

Example run:

```
Enter a temperature in Celsius: 25
Enter a distance in kilometers: 10
Enter a weight in kilograms: 70

=== Conversion Summary ===
25.0 C is 77.0 F and 298.15 K
10.0 km is 6.21371 miles
70.0 kg is 154.3234 lb
```

## How to run it

```bash
python3 starter.py
```

Fill in the `# TODO` sections in `starter.py`. Want to see one finished
version first? Run `python3 solution.py`.

## Ideas to make it your own (optional stretch goals)

## Why this project matters

Real-world scripts are rarely more complex than this at heart: take some
input, convert its type, do the math, present the result clearly. You'll
write this exact shape of program constantly in real jobs.
