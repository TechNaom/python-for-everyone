Chapter 2 Project 20 min build

Everyday Unit Converter

A genuinely useful little tool built from variables, types, and conversion — the kind of script you'd actually keep around.

Example run

Output 
Enter a temperature in Celsius: 25
Enter a distance in kilometers: 10
Enter a weight in kilograms: 70

=== Conversion Summary ===
25.0 C is 77.0 F and 298.15 K
10.0 km is 6.21371 miles
70.0 kg is 154.3234 lb
How to run Download starter.py below and run python3 starter.py.
starter.py solution.py (reference)

Ideas to make it your own

  • Round the output to 2 decimal places (a peek ahead — string formatting is covered properly in Chapter 6).
  • Add a currency conversion using a fixed exchange rate.
  • Add a second conversion: Fahrenheit back to Celsius.

Why this project matters

Real-world scripts are rarely more complex than this at heart: take some input, convert its type, do the math, present the result clearly. You'll write this exact shape of program constantly in real jobs.
