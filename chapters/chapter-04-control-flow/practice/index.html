Practice Bank: Control Flow

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Still uses only what Chapters 1-4 covered: print(), variables, input(), the conversion functions, arithmetic / relational / logical / bitwise operators, is / in, and if / if-else / elif chains / nested if.

On this page 1. Simple if 2. if-else 3. elif chains 4. Nested if
