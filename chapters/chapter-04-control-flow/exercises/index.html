🐞 Task 4 — Debug the Code

This elif chain is supposed to grade 90 as "A" but always prints "D". Find and fix the ordering bug.

buggy.py 
score = 90
if score >= 40:
    grade = "D"
elif score >= 50:
    grade = "C"
elif score >= 60:
    grade = "B"
elif score >= 75:
    grade = "A"
else:
    grade = "Fail"
print("Grade:", grade)
