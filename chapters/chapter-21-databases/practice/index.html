Practice Bank: Working with Databases

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Every "database" here is a FakeConnection/FakeCursor or FakeCollection class standing in for a real mysql-connector-python/pymongo object — no live database or installed driver is needed anywhere in this bank. No import beyond json/os/re/math/datetime/random/csv/threading/concurrent.futures.

On this page 1. What a Connection Is & Why a Driver 2. Connecting to MySQL & Cursors 3. Parameterized Queries & SQL Injection 4. MongoDB & Documents 5. CRUD with pymongo 6. Relational vs. Document Data 7. Cloud-Hosted Connections
