Practice Bank: Git Fundamentals

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. This chapter teaches git itself, not just Python syntax, so every problem here is a Python function that simulates or reasons about a git concept using plain data standing in for a repo's history. Standard library only — no installs needed.

Why simulate git instead of running it? This site runs your code in a browser sandbox with no real git installed, so every problem models a git concept as a Python function over plain data (lists of commit ids, booleans for branch state, strings for file content) instead of literally shelling out to git. Every behavior these problems describe was checked against real git in a scratch repository first — the semantics are genuinely correct, just expressed in Python so they can be auto-graded. To practice the real commands, install git and follow along with this chapter's lesson in an actual terminal.
