Chapter 26 Practice Bank (Categories 1-8) 90-120 min, hands-on

Practice Bank: Debugging Production Issues

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Topics 1-5 cover Categories 1-2 (Crashes & Exceptions, Performance); Topics 6-10 cover Categories 3-8 (Memory Leaks, Concurrency & Deadlocks, Data Corruption, Deployment & Environment, Database & API Failures, and Logging & Observability). Standard library only — json, copy, math, collections — no installs needed.

On this page 1. Missing Keys & Out-of-Range Access 2. Exception Handling That Hides Bugs 3. None, Zero & Other Quiet Failure Values 4. Accidental O(n^2) Patterns 5. Caching & Redundant Work 6. Unbounded Growth & Shared State 7. Concurrency Shapes 8. Data Corruption & Precision 9. Deployment & Environment 10. Database & API Reliability, and Observability
