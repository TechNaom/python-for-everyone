Chapter 13 Practice Bank 70-90 min, hands-on

Practice Bank: File Handling & CSV

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Uses everything from Chapters 1-12 plus this chapter's own toolkit: open()/file I/O with with, .read()/.readline()/.readlines(), write ("w") and append ("a") modes, "r+", relative vs. absolute paths, the csv module (csv.reader/csv.writer/ csv.DictReader/csv.DictWriter), and FileNotFoundError/PermissionError handling. No other class besides a custom exception subclassing Exception, and no import beyond math/datetime/random/csv.

A note on the sample files Every sample data file these problems read from (notes.txt, settings.txt, sales.csv, sales_extra.csv, app.log, orders.csv) lives in this same practice/ folder. Download them alongside starter.py and run everything from inside this folder — a plain relative filename like "notes.txt" only resolves correctly when Python is run from the same folder the file lives in.
On this page 1. Opening & Reading Files 2. Writing to Files 3. File Modes & Paths 4. The csv Module 5. Handling File-Related Exceptions 6. Bringing It Together
How to run these Download starter.py and the sample data files below into one folder, then run python3 starter.py from inside that same folder.
