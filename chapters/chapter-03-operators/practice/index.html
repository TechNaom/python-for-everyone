Chapter 3 Practice Bank 35-45 min, hands-on

Practice Bank: Operators

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Still uses only what Chapters 1-3 covered: print(), variables, basic types, and this chapter's arithmetic/assignment, relational/logical, bitwise, and is/in operators. No if, loops, or functions yet — conditions are just printed, not used to branch.

On this page 1. Arithmetic & assignment 2. Relational & logical 3. Bitwise 4. Special: is / in
How to run these Download starter.py below, then run it from a terminal with python3 starter.py.
starter.py solution.py (reference)

Topic 1: Arithmetic & assignment

1 All seven arithmetic operators

Find # TODO 1.1. Given a = 17 and b = 5, print the result of +, -, *, /, //, %, and **.

2 Every compound assignment shortcut

3 Celsius to Fahrenheit

4 Debug the Code

5 Area and perimeter

Find # TODO 1.5. Given length = 8 and width = 3, compute and print the area and perimeter of the rectangle.

Grocery bill split
