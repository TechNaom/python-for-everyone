Practice Bank: Multithreading

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Every "wait" here uses time.sleep() to stand in for a slow network or disk operation — no real network access or file I/O happens anywhere in this bank. No import beyond json/os/re/math/datetime/random/csv/threading/concurrent.futures.

On this page 1. What a Thread Is 2. Creating & Starting Threads 3. The GIL 4. Race Conditions 5. threading.Lock 6. Thread Safety Patterns 7. ThreadPoolExecutor
