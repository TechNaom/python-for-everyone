Chapter 19 Practice Bank 70-90 min, hands-on

Practice Bank: Working with APIs & JSON

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Every "response" here is a hardcoded JSON string standing in for what a real API would send back — no network access is needed or used anywhere in this bank. No import beyond json/os/re/math/datetime/random/csv.

On this page 1. What an API Is, and Why JSON 2. requests Basics 3. json Module 4. Navigating Nested JSON 5. Error Handling 6. Params & Headers 7. Fetch-Parse-Format Pipeline
How to run these Download starter.py below and run python3 starter.py from inside the same folder.
