Chapter 11 Practice Bank 60-80 min, hands-on

Practice Bank: Modules & Packages

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Uses everything from Chapters 1-10 plus this chapter's own toolkit: import x, from x import y, import x as y, from x import y as z, and the math, datetime, and random modules. No try/except, file I/O, classes, or any import beyond math/datetime/random.

A note on randomness and dates Any problem using random or datetime.date.today() prints a different result every time you run it — that's the point. solution.py checks a property of the result (like "is this in range") instead of one exact value, so your output won't match anyone else's line-for-line, and that's fine.
