Chapter 22 Practice Bank 70-90 min, hands-on

Practice Bank: Building Web Apps & APIs — Flask & FastAPI

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Every task here is a plain Python function or class modeling exactly what a real Flask or FastAPI route function does with its inputs — no live server or installed framework is needed anywhere in this bank. No import beyond json/os/re/math/datetime/random/csv/threading/concurrent.futures.

On this page 1. Web Framework & Request/Response Cycle 2. Flask Routes & render_template 3. The Flask Request Object 4. FastAPI Path & Query Params 5. Pydantic Models & Automatic Docs 6. Flask vs. FastAPI, Side by Side 7. What's Next: HTML, JSON & React
