Practice Bank: GitHub & Collaboration

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Standard library only, no installs needed. Since these run in a Python grader (not a real terminal), every problem reasons about or simulates a GitHub collaboration concept as a plain Python function — but the underlying git/GitHub behavior behind each one was verified against a real local bare-repo simulation first.

On this page 1. Remotes (origin, push/pull/fetch, tracking branches) 2. Forking vs. Cloning 3. Pull Requests 4. Code Review 5. Issues & Project Workflow 6. Real-World End-to-End Collaboration
How to run these Download starter.py below and run python3 starter.py from inside the same folder. Standard library only — nothing to install.
