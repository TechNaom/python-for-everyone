Practice Bank: Loops (while & for)

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Loops are one of the most common interview warm-up areas, so several of these lean directly into that. Still uses only while loops, for loops with range(), the accumulator pattern, break/continue, and everything from earlier chapters.

On this page 1. while loops 2. for loops and range() 3. Common loop patterns 4. break & continue
How to run these Download starter.py below, then run it from a terminal with python3 starter.py.
