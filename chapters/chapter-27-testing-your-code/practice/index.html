Practice Bank: Testing Your Code

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. unittest and unittest.mock are standard library, no installs needed; the pytest topic returns pytest-style source as strings so the file still runs standalone.

On this page 1. Why Testing Matters 2. Writing Tests with unittest 3. Writing Tests with pytest 4. Fixtures and setUp/tearDown 5. Mocking and Test Doubles 6. Test Coverage & What Good Tests Look Like
How to run these Download starter.py below and run python3 starter.py from inside the same folder. unittest/unittest.mock are standard library — nothing to install for those. If you want to actually execute the pytest source strings from Topic 3, save them to a file and pip install pytest first.
