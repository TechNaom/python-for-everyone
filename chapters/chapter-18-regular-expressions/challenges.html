1 Valid Email Checker Easy

Write a function is_valid_email(text) that returns whether text, in its entirety, looks like a valid email address: one or more letters/digits/./+/-/_, then an @, then one or more letters/digits/-, then a literal dot, then 2 or more letters. Use re.fullmatch() with the pattern r"[\w.+-]+@[\w-]+\.[A-Za-z]{2,}" and return whether the result is not None.

Constraints: text is always a string, possibly empty.
Examples:
is_valid_email("ann@example.com") → true
is_valid_email("not-an-email") → false
challenge_1.py
Show Solution
solution_1.py 
import re

def is_valid_email(text):
    return re.fullmatch(r"[\w.+-]+@[\w-]+\.[A-Za-z]{2,}", text) is not None

7 Parse a Log Line Into Named Fields Medium

Write a function parse_log_line(line) that matches line against the pattern r"(?P<time>\d{2}:\d{2}:\d{2})\s+\[(?P<level>\w+)\]\s+(?P<message>.+)" using re.match(). If it matches, return match.groupdict() (a dict with keys "time", "level", and "message"). If it doesn't match, return None.

Constraints: line may not be in the expected log format at all, in which case return None.
Examples:
parse_log_line("10:15:00 [ERROR] disk full") → {"time": "10:15:00", "level": "ERROR", "message": "disk full"}
parse_log_line("not a log line") → null
challenge_7.py
Show Solution
solution_7.py 
import re

def parse_log_line(line):
    match = re.match(
        r"(?P\d{2}:\d{2}:\d{2})\s+\[(?P\w+)\]\s+(?P.+)",
        line,
    )
    if match is None:
        return None
    return match.groupdict()

8 First N Matches, Using a Compiled Pattern Medium

Write a function first_n_matches(text, pattern, n) that builds a compiled pattern with re.compile(pattern), then loops over compiled.finditer(text), collecting each match's .group() into a list, and stopping (breaking out of the loop) once the list already has n items — even if more matches remain in text. Return the list.

Constraints: there may be fewer than n matches total in text, in which case return only however many were actually found.
Examples:
first_n_matches("a1 b2 c3 d4 e5", r"[a-z]\d", 3) → ["a1", "b2", "c3"]
first_n_matches("no matches", r"\d+", 2) → []
challenge_8.py
Show Solution
solution_8.py 
import re

def first_n_matches(text, pattern, n):
    compiled = re.compile(pattern)
    results = []
    for match in compiled.finditer(text):
        if len(results) >= n:
            break
        results.append(match.group())
    return results
