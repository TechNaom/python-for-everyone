Chapter 14 Practice Bank 70-90 min, hands-on

Practice Bank: OOP Basics

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. This is the chapter where general classes become allowed: everything from Chapters 1-13 (including custom exception classes) plus class definitions, __init__, instance attributes, class attributes, instance methods, and __str__/__repr__. No @staticmethod, @classmethod, @property (Chapter 15), inheritance (Chapter 16), or any import beyond math/datetime/random/csv.

On this page 1. Classes & Objects 2. __init__ and Instance Attributes 3. Instance Methods & self 4. Class Attributes vs. Instance Attributes 5. __str__ and __repr__ 6. Bringing It Together
