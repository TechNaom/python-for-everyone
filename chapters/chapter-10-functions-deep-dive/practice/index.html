Chapter 10 Practice Bank 75-95 min, hands-on

Practice Bank: Functions Deep Dive

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. Uses everything from Chapters 1-9 plus this chapter's own toolkit: def, parameters and default arguments, *args and **kwargs, local/global scope with global/nonlocal, recursion, closures, and decorators (hand-written, no functools). No try/except, file I/O, classes, or imports.

On this page 1. Function Basics 2. Default Arguments 3. *args & **kwargs 4. Scope 5. Recursion 6. Closures 7. Decorators 8. Bringing It Together
How to run these Download starter.py below, then run it from a terminal with python3 starter.py.
