Chapter 31 Practice Bank 70-90 min, hands-on

Practice Bank: Professional Python

A deeper set of practice problems, organized by topic, for extra reps beyond the main exercises — including scenario-based problems written in the same style you'll meet in real interviews. This chapter covers venv, project structure, logging, argparse, packaging basics, and shipping a real CLI tool.

Why some of these are "reasoning" problems Virtual environments, project structure, and packaging are mostly command-line and config-file skills, not Python logic — so those problems are Python functions that simulate or reason about a described venv/pip/pyproject state using plain data (strings, lists, dicts) instead of literally running `pip` or reading a real `pyproject.toml`. Every claim they encode was checked against real venv/pip/Python behavior in a scratch directory first. Logging and argparse, by contrast, are real, runnable Python using the standard library — write and run actual code for those.
On this page 1. Virtual Environments 2. Project Structure 3. Logging 4. argparse 5. Packaging Basics 6. Putting It Together
How to run these Download starter.py below and run python3 starter.py from inside the same folder. Standard library only — nothing to install.
